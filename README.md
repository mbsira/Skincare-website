# LUMIÈRE | Skincare Website

A luxury skincare e-commerce landing page built with HTML, CSS, and vanilla JavaScript.

![LUMIÈRE Preview](skincare.png)

## Features
- Full-screen video hero section
- Working cart sidebar with add/remove/quantity
- Animated marquee banner
- Shop by skin concern section
- Product grid with hover interactions
- Promo band, about section with video
- Category list, contact form
- Toast notifications
- Fully responsive

## Tech Stack
- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript

## Run Locally
Open with Live Server in VS Code or run:
```bash
python3 -m http.server 8080
```
Then visit `http://localhost:8080`

## Files
- `index.html` — markup
- `style.css` — all styles
- `script.js` — cart logic and interactions
